What is DOM in javascript ?

DOM-document Object Model. DOM হচ্ছে HTML document এর জন্যে একটা প্রোগ্রামিং ইন্টারফেস । যখন ওয়েবপেজ লোড হয় তখন HTML document টা DOM Tree তে convert হয়। DOM এর দ্বারাই javascript খুব easily HTML element গুলো কে manipulate করতে হয়

DOM ব্যবহার করে javascript কি কি করতে পারে ?

  1. HTML elements গুলোকে পরির্বতন করতে পারে
  2. HTML elements গুলোকে CSS style পরির্বতন করতে পারে
  3. নতুন HTML element ক্রিয়েট করতে পারে
  4. HTML elements গুলোকে রিমুভ করতে পারে
  5. HTML attributes গুলোকে পরির্বতন করতে পারে
  6. নতুন HTML events ক্রিয়েট করতে পয়ারে

Node বলতে কি বোঝায় ?

Nodes দিয়ে DOM Tree বানানো হয়। Node হচ্ছে objects , DOM Tree তে Document , Doctype , elements , text গুলোকে nodes বলে।

parentNode এবং childNode বলতে কি বোঝায় ?

Root Node ছাড়া প্রতিটি নোডের একটি প্যারেন্ট নোড থাকে একটা parent node এর একাধিক child node থাকতে পারে সেম parent এর child node গুলোকে neighbounrs বা sibling nodes বলা যায়।

DOM এর methods বলতে কি বোঝায় ?

DOM methods দ্বারা আমরা javascript দিয়ে HTML element গুলোকে manipulate (add , delete , create , remove , modify etc) করতে পারি ।

  1. getElementById: এই method Id দিয়ে একটি elements কে access করা
  2. getElementByclassName: একটি নিদিষ্ট class name এর উপর ভিত্তি করে elements গুলোকে access করে।
  3. getElementByTagName: একটি নিদিষ্ট tag এর উপর ভিত্তি করে elements গুলোকে access করে।
  4. getElementByTagNameAll: একটি নিদিষ্ট tag এর উপর ভিত্তি করে ওই সকল tag elements গুলোকে access করে।
